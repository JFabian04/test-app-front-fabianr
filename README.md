# App

Aplicación frontend construida con Next.js utilizando App Router y una arquitectura feature-based.  
El proyecto está orientado a dashboards administrativos y módulos CRUD escalables.

## Requisitos

- Node.js 18 o superior
- npm 

## Instalación

1. Instalar dependencias:

```bash
npm install
```

2. Copiar el archivo de variables de entorno:

```bash
cp .env.example .env.local
```

3. Ejecutar el servidor de desarrollo:

```bash
npm run dev
```

La aplicación estará disponible en http://localhost:3001
